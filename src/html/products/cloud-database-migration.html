云数据库迁移

在云上创建数据库后，您经常面临云下数据库迁移上云的场景。通过数据复制服务，您能够简单、便捷地完成云下数据库迁移上云。在迁移过程中，数据复制服务能够最大限度地缩短系统业务中断时间，降低对业务影响。

产品介绍

  1. 在应用和系统迁移的过程中，源机无需停止数据库；业务不受影响。支持本地或者长距离远程迁移。

  2. 支持在不同的硬件平台之间进行 Oracle 数据库迁移。

  3. Windows支持Windows 2003、2008、2012，linux上的oracle在线云迁移。

应用场景

通过使用数据复制服务，您可以实现云下到云上、跨云平台、VPC网络内，以及云内到云外等多种业务场景下的数据库迁移。
